# 数据状态划分
1. 原始数据 未补充信息
2. 原始数据 已补充信息
3. 数据初筛(去重，剔除已有数据)
4. 筛查验证后的数据(通过天眼查，百度等方式进行筛查)
5. 入库数据(进行去重)
6. 未发送短信数据
7. 已经发送过短信的数据_未有回复
8. 已经发送过短信的数据_已有回复
9. 无效数据

    * 无效数据库
    * 未经过验证的数据库
    * 经过验证的数据库
    * 已发送短信的数据库
